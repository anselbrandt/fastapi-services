# FastAPI Services

### Docker

```
cd backend

docker compose up -d --build
```

http://localhost:8000

### Install

Requires [UV package and project manager](https://docs.astral.sh/uv/getting-started/installation/) for Python

```
cd backend

uv sync

uv run task dev

or

uv run task start
```
